1.📌 Descripción

Este proyecto consiste en una página web que permite calcular el total de compra de un producto, modificando la cantidad mediante botones.

🛠 Tecnologías utilizadas
HTML
CSS
JavaScript

⚙️ Funcionalidad
Aumentar y disminuir el precio por la cantidad de productos.
Calcular automáticamente el total a pagar.
